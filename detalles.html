¿Que métodos de pago disponibles hay?

Actualmente el método más rápido y fiable es por PayPal, pero también se acepta transferencias bancarias o Bizum.

¿Cuál es el plazo de pago?

El pago se hará cuando el cliente vea y apruebe el resultado final del proyecto. Hasta que no se finalice por completo no se realizará ningún pago, así se garantiza comodidad al cliente.

¿Cómo es el proceso de entrega?

Lo primero que se empieza a hacer es pedir información al cliente sobre su idea, se hacen varias reuniones mediante videoconferencias. Una vez recogida la información necesaria se empiezan a crear varios bocetos e ideas que se les mostrarán al cliente, es el cliente el que decide lo que hacer y el programador es el que le da ideas y consejos de lo que necesite. Para finalizar, una vez que se elija el diseño que más guste se comenzará a programar y finalizar el proyecto.
Cualquier documento/archivo que se necesite o que se le vaya a entregar al cliente se hará mediante correo electrónico.

¿Cuánto tiempo se tarda?

El tiempo varía mucho dependiendo del proyecto, si es muy grande o algo más pequeño, si hay que recaudar mucha información o no es tanta, si el cliente no queda satisfecho y hay que realizar varias pruebas... Por ello siempre se saca una aproximación de lo que pude tardarse pero no siempre es ese tiempo. También si se necesita se puede poner un plazo de entrega máximo por si hay prisa.

¿Como puedo saber si es fiable?

Cuando se contrata alguna de los servicios se mantendrá informado al cliente en todo momento de sus avances, también se le mostrará todo sin ocultar nada. A su vez se harán de vez en cuando algunas videoconferencias para comentar lo que se necesite.

¿Que pasa con mis datos personales?

Los datos recogidos y que se utilicen solo estarán públicos para el cliente y el programador, y no se usará cualquier dato/documento para nada salvo el proyecto. Una vez finalizado el trabajo el programador ya no se hará cargo de los datos excepto algunos servicios que lo requieran.
